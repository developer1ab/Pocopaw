# pocopaw Showcases

This section is the public demo gallery for reproducible product flows.

Each showcase should include:

1. A short demo video (30-90 seconds)
2. A clear scenario and user goal
3. Preconditions (device, permission state, network)
4. Step-by-step execution notes
5. Result and measurable outcomes
6. Reproduction instructions

## Quick Start

1. Copy [showcase-template.md](showcase-template.md)
2. Rename it to a new file, for example `task-execution-closed-loop.md`
3. Put your video under `public-docs/showcases/`
4. Link the video from the showcase file
5. Add the showcase file link to the list below
